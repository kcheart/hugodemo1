# 13. 使用 GitHub Destop 上傳到 GitHub 網站

(看教師電腦的廣播畫面操作)
打開 GitHub Desktop
選擇倒三角形符號，點選 `Add`
加入已經存在的 repository: `Add existing repository...`
選擇 `hugodemo`
點選 `Add repository`

First Commit
Commit to master

Publish your repository to GitHub
點選 `Publish repository`
注意： 把 Keep this code private 的勾勾取消掉
